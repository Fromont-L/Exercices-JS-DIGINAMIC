  • Les Tableaux

    • let chaines = ['truc0', 'truc1', 'truc2', 'truc3', 'truc4']; console.log(chaines.length); ça affiche le nombre de cases. console.log(chaines[5]); ça fait une erreur car le 6eme élément du tableau n'existe pas console.log(chaines[chaines.lenght-1]); permet de parcourir toutes les cases chaines.push("ajouté après le dernier élément") permet d'ajouter chaines.splice(1,1) permet de supprimer 1 élément dans la liste à la 2ème position chaines.slice() permet de dupliquer un tableau mais garde les valeurs à partir du moment où il est déclaré, n'affecte pas les futurs changements
